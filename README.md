# bibliotheksverwaltung

## How To Contribute

- Fork erstellen
- Fork klonen
- Branch erstellen
- Änderungen Vornehmen
- Commits erstellen
  - **Sinnvolle** Namen & Beschreibung wählen
  - Ggf. issues markieren
  - Keine irrelevanten Änderung in commits
- Commits pushen
- Pull Request erstellen
  - **Sinnvolle** Namen & Beschreibung wählen
